*مینی شاپ*
پروژه تمرینی مینی شاپ با استفاده از ریکت ساخته شده و از مفاهیم redux استفاده شده

تکنولوژی ها : 
1 html 
2 css 
3 javaScript
4 react


توضیحات : 
این پروژه تمرینی ریداکس من هست که از مفاهیم پایه هم توش استفاده شده .
در پروژه محصولات از قبل گذاشته شده با استفاده از api در محل صفحه اصلی نمایش داده شده اند .
شما می توانید با ساختن اکانت و یا لاگین کردن اکانت خود در سایت محصولاتی که دوست دارید بسازید .
شما یک داشبورد دارید که می توانید اکانت خود را مشاهده کنید و کار های غیره انجام دهید 

ویژگی ها: 
1 کامپوننت جداگانه برای هر عملیات 
2 ساختار دهی درست 
3 مدیریت پیشرفته state ها 


معایب : 
1 پروژه جای بهبود زیادی دارد
2 نداشتن فایل css
